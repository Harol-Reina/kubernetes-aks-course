# Lab 01: Application Troubleshooting - Setup Scripts

Este laboratorio requiere que apliques los manifests con errores y los diagnostiques.

## Archivos Disponibles

- `README.md` - Instrucciones completas del lab
- `setup-scenarios.sh` - Script para crear todos los escenarios
- `cleanup.sh` - Script de limpieza

## Uso

```bash
# Crear todos los escenarios
chmod +x setup-scenarios.sh
./setup-scenarios.sh

# Trabajar en cada escenario (ver README.md)

# Limpiar al finalizar
chmod +x cleanup.sh
./cleanup.sh
```

Los manifests están embebidos en el README.md en cada escenario. Copia y aplica según sea necesario.
